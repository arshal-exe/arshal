# Google Search Optimization Recommendations

Checklist to improve Google discoverability for a personal portfolio:

1. Semantic HTML: use proper heading hierarchy (H1 per page), landmarks (`<main>`, `<nav>`, `<footer>`).
2. Titles & Descriptions: unique `<title>` and `<meta name="description">` for each page.
3. Structured Data: add `Person` schema JSON-LD.
4. Sitemap & robots.txt: include and register sitemap in Google Search Console.
5. Image optimization: responsive `srcset`, WebP and proper `alt` attributes.
6. Performance: serve compressed assets, enable caching, minimize render-blocking CSS/JS.
7. Accessibility: ensure keyboard navigation and ARIA attributes; accessibility improvements have SEO benefits.
8. Mobile-first: ensure responsive breakpoints and fast mobile load times.
9. Canonical URLs: add `<link rel="canonical">` to avoid duplicate content.
10. Analytics / Search Console: add Google Analytics (or alternative) and register site with Search Console.
