# Social Preview Image Recommendation

Create a 1200×630px social preview image for Open Graph and Twitter that matches your site's visual style.

- Background: very dark radial gradient (match main site: deep green/black)
- Accent: neon green (#00ff00) and warm orange (#ff7a00) accents
- Typography: bold geometric sans for name (large), smaller mono/subhead in `Courier New` style for role
- Layout: left-aligned name and role, right-aligned circular avatar/logo
- Icons: small GitHub / Envelope icons near bottom as subtle indicators
- File format: export as PNG or WebP, keep filesize < 200 KB

Filename suggestion: `social-preview.png` and reference it in Open Graph meta tags.
